# ColorPallete

Convey meaning through color. ColorPalette is intended to display one color hue with their shades.

## Variants

### Default

<Preview>
  <ColorPallete name="primary" color="#ff5722" />
</Preview>

### Multiple shades

<Preview>
  <ColorPallete
    name="error"
    color={{
      '300': '#ff6b6b',
      '500': '#fa5252',
    }}
  />
</Preview>
