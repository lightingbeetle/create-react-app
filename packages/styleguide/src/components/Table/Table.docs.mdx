# Table

Use the table component to make information easier to compare and scan for users.

## Variants

### Default

<Preview>
  <Table
    columns={[
      { key: 'first', label: 'First' },
      { key: 'second', label: 'Second' }
    ]}
    data={[
      {
        first: 'First item in first column',
        second: 'Second item in first column'
      },
      {
        first: 'First item in second column',
        second: 'Second item in second column'
      }
    ]}
  />
</Preview>

### DocsTable

Table with predefined columns to describe components props or every typical config.

<Preview>
  <DocsTable
    data={[
      {
        prop: 'prop1',
        type: 'string',
        default: 'default',
        description: 'This is prop description',
      },
      {
        prop: 'prop2',
        type: 'func',
        default: '(attr) => {}',
        description: 'This is prop description',
      },
    ]}
  />
</Preview>
